Platform notes
ChatGPT

Append "Show your sources" to your query to ensure sources appear. Click the Sources button at the bottom of the answer to open the panel before running the bookmarklet.

Perplexity

Switch to the Links tab (in the tab bar above the answer) before running the bookmarklet. The Links tab contains real URLs; the Answer tab's inline citations are text-only and don't contain extractable links.

Google AI Mode

Click to expand the source cards (the "X sites" indicator), then click Show all if present before running the bookmarklet.

Incognito setup

Bookmarklets work in incognito by default

You just need the bookmarks bar visible. Open an incognito window and press Ctrl+Shift+B (Windows) or ⌘+Shift+B (Mac) to toggle it on.

Troubleshooting

0 domains found

Make sure the source list panel is fully open and visible. Some platforms lazy-load sources — scroll to the bottom of the source list to ensure all items are rendered.

Clipboard not working

Some browsers block clipboard access on certain pages. If this happens, a fallback popup will appear with the domain list for manual copying.

Missing some domains

The platforms update their page structure frequently. If the bookmarklet consistently misses sources on a specific platform, the DOM selectors may need updating.
